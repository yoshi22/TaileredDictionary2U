# Assets

Add the following image files to this directory:

## Required Assets

- **icon.png** (1024x1024): App icon for iOS and Android
- **splash.png** (1284x2778 recommended): Splash screen image
- **adaptive-icon.png** (1024x1024): Android adaptive icon foreground

## Optional

- **notification-icon.png** (96x96): Push notification icon (Android)
- **favicon.png** (48x48): Web favicon

## Design Guidelines

- Use PNG format with transparency
- App icon should have rounded corners (iOS will apply automatically)
- Splash screen should work with #3B82F6 (blue) background color
- Adaptive icon should have transparent background

## Temporary Development

During development, you can use any placeholder images. The app will work with default Expo icons until you add custom assets.
